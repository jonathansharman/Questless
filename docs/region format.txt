Hex types
	0 - edge - black, impassable
	1 - rock
	2 - dirt
	3 - grass
	4 - road - buff speed
	5 - tree - debuff speed
	
			
Example 4 x 4 block file:

4 4
0 0 0 0
0 1,2,5 1,3 0
0 1,3 1,2 0
0 0 0 0
